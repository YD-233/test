# Rshell - 跨平台多协议C2框架

Rshell是一款开源的golang编写的支持多平台的C2框架，旨在帮助安服人员渗透测试、红蓝对抗。功能介绍如下：

## 基础使用

下载Rshell二进制文件并运行。

通过-p参数指定端口（默认端口8089）并运行：

```
./Rshell -p 8089
```

**默认账号密码：admin/admin123**
